# Trademark Usage Guide

This document ensures consistent use of trademark symbols (™) throughout the WunderBrand Snapshot™ system.

## Trademarked Terms

Always include ™ symbol for:

- **WunderBrand Snapshot™** (not "WunderBrand Snapshot™" or "brand snapshot")
- **WunderBrand Score™** (not "WunderBrand Score™" or "brand alignment score")
- **Snapshot+™** (not "Snapshot+" or "snapshot+")
- **Blueprint™** (not "Blueprint" or "blueprint")
- **Blueprint+™** (not "Blueprint+" or "blueprint+")

## Usage Examples

### Correct ✅
- "Your WunderBrand Snapshot™ is ready"
- "WunderBrand Score™: 78"
- "Upgrade to Snapshot+™"
- "Generated by Wundy™ • WunderBrand Snapshot™ • Wunderbar Digital"

### Incorrect ❌
- "Your WunderBrand Snapshot™ is ready"
- "WunderBrand Score™: 78"
- "Upgrade to Snapshot+"
- "Generated by Wundy™ • WunderBrand Snapshot™ • Wunderbar Digital"

## ActiveCampaign Merge Tags

When using in ActiveCampaign email templates, include ™ in merge tag names:

- `%BRAND_ALIGNMENT_SCORE%` (field name: "WunderBrand Score™")
- `%SNAPSHOT_PLUS_PITCH%` (field name: "Snapshot+™ Upsell Block")

## HTML Entity

In HTML, use: `&trade;` or `™`

In React/JSX, use: `™` (Unicode character)
