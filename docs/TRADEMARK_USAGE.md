# Trademark Usage Guide

This document ensures consistent use of trademark symbols (™) throughout the Brand Snapshot system.

## Trademarked Terms

Always include ™ symbol for:

- **Brand Snapshot™** (not "Brand Snapshot" or "brand snapshot")
- **Brand Alignment Score™** (not "Brand Alignment Score" or "brand alignment score")
- **Snapshot+™** (not "Snapshot+" or "snapshot+")
- **Blueprint™** (not "Blueprint" or "blueprint")
- **Blueprint+™** (not "Blueprint+" or "blueprint+")

## Usage Examples

### Correct ✅
- "Your Brand Snapshot™ is ready"
- "Brand Alignment Score™: 78"
- "Upgrade to Snapshot+™"
- "Generated by Wundy • Brand Snapshot™ • Wunderbar Digital"

### Incorrect ❌
- "Your Brand Snapshot is ready"
- "Brand Alignment Score: 78"
- "Upgrade to Snapshot+"
- "Generated by Wundy • Brand Snapshot • Wunderbar Digital"

- `app/api/og/report/route.ts` - OG image
- `COMPLETE_HOME_PAGE.html` - Parent page
- All email templates in ActiveCampaign

## ActiveCampaign Merge Tags

When using in ActiveCampaign email templates, include ™ in merge tag names:

- `%BRAND_ALIGNMENT_SCORE%` (field name: "Brand Alignment Score™")
- `%SNAPSHOT_PLUS_PITCH%` (field name: "Snapshot+™ Upsell Block")

## HTML Entity

In HTML, use: `&trade;` or `™`

In React/JSX, use: `™` (Unicode character)
