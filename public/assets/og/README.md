# OG Assets

This folder contains brand assets used by Open Graph routes under `app/api/og/*`.

If you have finalized PNGs from design, replace the placeholder SVGs with the requested PNG filenames:
- `brand-snapshot-bg.png`
- `snapshot-plus-bg.png`
- `blueprint-bg.png`
- `blueprint-plus-bg.png`
- `logo-wunderbar.png`
- `wundy-outline.png`

For now, the OG routes use the SVG placeholders in this directory.
